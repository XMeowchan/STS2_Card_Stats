# STS2 卡牌统计悬浮窗

这是一个给《Slay the Spire 2》使用的 Mod。

安装后，把鼠标移到卡牌上时，卡牌旁边会额外显示一块统计面板，直接展示这张卡的大盘参考数据，帮助你更快做出选牌判断。

## 它能帮你做什么

- 在游戏里直接看卡牌统计，不用来回切网页
- 更快判断一张卡值不值得拿
- 作为构筑和路线决策时的额外参考

这些数据是社区统计，仅供参考，不会改动卡牌效果、战斗流程或存档。

## 安装方法

推荐直接使用发布页里的安装包：

1. 打开 [Releases](https://github.com/XMeowchan/STS2_Card_Stats/releases)
2. 下载最新的 `HeyboxCardStatsOverlay-Setup-版本号.exe`
3. 先关闭游戏
4. 双击运行安装包
5. 安装完成后启动《Slay the Spire 2》

安装程序会自动把 Mod 放进游戏的 `mods` 文件夹。

## 手动安装

如果你拿到的是解压后的文件，而不是安装器：

1. 找到《Slay the Spire 2》游戏目录下的 `mods` 文件夹
2. 把整个 `HeyboxCardStatsOverlay` 文件夹复制进去
3. 启动游戏

最终目录应类似：

```text
Slay the Spire 2\mods\HeyboxCardStatsOverlay
```

## 怎么使用

安装成功后不需要额外设置：

1. 进入游戏
2. 把鼠标悬停到卡牌上
3. 在卡牌旁边查看统计面板

## 适合谁

如果你想在打牌时更快获得卡牌参考信息，而不是频繁切网页或查表，这个 Mod 就很适合你。
